Raytracer

I wrote a raytracing program from scratch that reads in a custom input file describing the entire scene. This includes the location of the ‚Äúeye‚Äù, however many lights are defined, and however many spheres are defined. This also includes the colors of those defined spheres, the field of view, the size of the image, the background color, the view direction, and the up direction. My program takes in all the definitions of these things and determines the color to show at each pixel of the output image.

  • 500+ lines of C++ code!
  • Reads in a fully customizable scene input file!
  • Renders output ASCII PPM images!
  • Example input PPM files in the Github link!

Space Minesweeper

Do you like those retro space shooter games? I made one accessible right in your browser! Fly around in a little ship controlled by your mouse and click to shoot lasers! Rendering With Shaders

This GLSL shader program allows the user to test out different shading styles on some 3D objects. The program uses per-pixel lighting and normals to create this effect. Users can select the shading style and object from the top-right menu. The main three shading styles include are Gouraud, Phong, and a Legend of Zelda inspired cartoon style. The yellow circle in the scene represents our light source and it can be set to be a point light or directional light. Users can click and drag to spin the object and the light. Want to test it out? Click the link below!

  • Written in TypeScript!
  • Many object options!
  • Features Gouraud, Phong, and Cartoon style shading!
  • Light can be set to point or directional!
  • Check out the working demonstration!
